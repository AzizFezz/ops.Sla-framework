Vous trouverez les étapes d'installation du projet dans le fichier commandes.

les fichiers de SLA Framework ainsi que le Dockerfile pour créer votre image se trouvent dans le dossier sla-core.
l'image déjà réalisée correspond à l'archive slaf.tar. Il faudra la déployer dans minikube.
Les trois fichers deployment définissent les services à déployer sur minikube.

Il y a quelques exemples de requêtes API pour réaliser les communications avec l'API de SLA Framework.
